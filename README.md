# Satellite Visualization Dashboard

An interactive satellite visualization dashboard that tracks orbital motion and interference patterns to support low-Earth-orbit analysis and research.

Check out the tool at: https://huggingface.co/spaces/vartikaa/satellite-dashboard

## Features
- **Orbit visualization:** Plot satellite ground tracks and sky positions from TLE data
- **Interference insights:** Explore potential interference regions/patterns (when available in the dataset)
- **Interactive UI:** Filter by satellite/constellation, time window, and view mode
- **Export-friendly:** Save plots/screenshots for reports and debugging

## Tech Stack
- **Python**, **Streamlit**
- **Plotly** (interactive charts/maps)
- **Skyfield** (orbit propagation)
